# Unmasked (iOS )

## Project Description

Unmasked is an evolving mobile application based on a supehero themed api. It currently connects users to a plethora of Heroes and 
Villains across a multitude of Universes in one central place. The Apps User experience is centered around a game that tests character 
knowledge and facilitates the unlocking of characters in the users personal hero collection. 

## App User Features
- User profiles, with personal hero collections.
- Universal hero search function that allows users to browse all characters
- Hero Statistic View with a summary of individual hero stats
- Unmasked game that compares two characters against their stats

## Technologies Used
- RESTful Api handling (with use of The Superhero Api: https://www.superheroapi.com)
- App based on the MVVM Architecture
- Firebase Realtime Database
- Swift
- Objective C
